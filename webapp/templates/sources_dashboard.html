How to read this dashboard

Each source shows how many items it contributed at each stage: Collected (what the source returned) → In Window (matched your date range) → Relevant (matched health keywords) → Unique (after removing duplicates).

If a source shows 0 collected, it either wasn't accessible (check terminal logs) or returned nothing for that date. If it shows items collected but 0 relevant, nothing matched the health keyword list that day.
